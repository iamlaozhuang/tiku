## 一、学员端 (Mobile-first)

学员端以“移动端操作习惯”为核心，界面强调单列流式布局、大尺寸触摸热区。

### 1. 核心导航结构 (Tab Bar)

学员端底部默认配置 3 个主 Tab：

1. **首页 (Home)**：今日任务、进行中的考试、题库入口
2. **错题本 (Mistakes)**：客观题错题收集与重做
3. **我的 (Profile)**：授权信息、卡密兑换、个人设置

### 2. 关键页面线框

#### 2.1 模拟考试 - 答题页 (`/mock-exam/:id`)

**布局特点**：全屏沉浸式，无底部 Tab Bar。

```text
+--------------------------------------------------+
| < 返回 | [大题标题 - 如：理论知识选择题] | 交卷   |  <- Header 浮动固定
|--------------------------------------------------|
| [ 倒计时 Timer: 119:59 ] (黄/红变色预警)         |
|--------------------------------------------------|
|                                                  |
| 1. 这是题目的题干描述内容，支持富文本排版？      |  <- QuestionCard
|    ( ) A. 选项描述 A                             |
|    (x) B. 选项描述 B                             |
|    ( ) C. 选项描述 C                             |
|    ( ) D. 选项描述 D                             |
|                                                  |
|--------------------------------------------------|
| 2. 这是第二题的题干描述内容...                   |
| ...                                              |
|                                                  |
|--------------------------------------------------|
|    [ 🌟 AI 提示 ]  [ 下一题/上一题控制区 ]        |  <- 悬浮操作/卡片内嵌
+--------------------------------------------------+
| [ 📝 答题卡 (已答 12/50) ]                        |  <- 底部固定悬浮按钮/栏
+--------------------------------------------------+
```

#### 2.2 考试报告页 (`/report/:id`)

**布局特点**：注重结果展示的数据可视化。

```text
+--------------------------------------------------+

|--------------------------------------------------|
|                  [ 85 分 ]                       |  <- ReportScoreRing (醒目展示)

|--------------------------------------------------|
| [ 查看解析 (主观题 AI 评分详情) ] -> 跳转        |
+--------------------------------------------------+
```

#### 2.3 授权与兑换页 (`/profile/redeem`)

**布局特点**：表单中心化，简化输入障碍。

## 二、后台端 (Desktop-first)

后台端遵循 B 端管理系统规范，强调信息密度、复杂检索和批量操作。采用侧边栏（Sidebar）布局。

### 1. 核心导航结构 (Sidebar)

1. **Dashboard 首页**
2. **题库管理**：试卷管理、题目管理、材料管理
3. **用户管理**：学员管理、组织管理（企业层级）
4. **授权管理**：卡密生成/发放、订单/兑换记录
5. **系统设置**：管理员管理、日志管理、AI 模型配置

### 2. 关键页面线框

#### 2.1 试卷管理列表页 (`/admin/papers`)

**布局特点**：经典的顶部筛选 + 中间表格。

```text
+-----------------------------------------------------------------------------+
| [LOGO 中国烟草题库] | 面包屑：题库管理 / 试卷管理           | [👤 Admin ∨] |
+-----------------------------------------------------------------------------+
| [ 🏠 Dashboard ] | =========================================================|
| [ 📚 题库管理  ] | [新增试卷 (主按钮)]  [导入试卷]                            |
|    - 试卷管理    |                                                          |
|    - 题目管理    | 专业: [全部 ∨]  等级: [全部 ∨]  状态: [已发布 ∨]         |  <- FilterBar
| [ 👥 用户管理  ] | 搜索词: [输入试卷名称...]        [ 搜索 ] [ 重置 ]         |
| [ 🔑 授权管理  ] | -------------------------------------------------------- |
| [ ⚙️ 系统设置  ] | 试卷名称        | 专业 | 等级 | 题量 | 状态 | 操作       |  <- Table
|                  | -------------------------------------------------------- |
|                  | 2026理论模拟卷A | 营销 | 中级 | 50   | 🟢   | [编辑][下架]|

```
