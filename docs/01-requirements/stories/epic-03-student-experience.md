# Epic 03: 学员端答题与体验

> 源文档：[03-student-experience.md](../modules/03-student-experience.md)

---

## 1. 首页与导航

### US-03-01: 学员首页

**作为**学员，**我想要**登录后看到授权范围内的试卷列表，**以便**选择学习内容。

**验收标准**：

- [ ] AC-1: 展示有效授权范围内的专业/等级。
- [ ] AC-2: 多个授权时记住上次选择，首次引导选择。
- [ ] AC-3: 当前专业/等级下按科目（理论/技能）分组显示试卷。
- [ ] AC-4: 每套试卷提供"练习"和"模拟考试"两个入口。
- [ ] AC-5: 试卷列表按发布时间倒序。
- [ ] AC-6: 无授权用户直接进入卡密兑换页。

**优先级**：P0

---

## 2. 练习模式

### US-03-02: 理论卷练习

**作为**学员，**我想要**逐题练习理论卷并获得即时反馈，**以便**巩固知识。

**验收标准**：

- [ ] AC-1: 一题一页，答完即显示对错、正确答案、解析。
- [ ] AC-2: 答错自动进入错题本 + 自动生成 AI 讲解。
- [ ] AC-3: 答对时显示答案/解析 + "需要讲解"按钮。
- [ ] AC-4: 客观题不允许再答一次。

**依赖**：US-04-04（AI 讲解）

**优先级**：P0

---

### US-03-03: 技能卷练习

**作为**学员，**我想要**按材料题组练习技能卷，**以便**提升实操能力。

**验收标准**：

- [ ] AC-1: 按材料题组/大题一组一页，材料置顶可折叠/展开。
- [ ] AC-2: 主观题提交后可选"AI 提示并重答一次"或"直接查看评分"。
- [ ] AC-3: 最多 1 次重答，第二次后给出最终评分。
- [ ] AC-4: 主观题不进入错题本。

**依赖**：US-04-01（AI 评分），US-04-04（AI 提示）

**优先级**：P0

---

### US-03-04: 练习暂停与继续

**作为**学员，**我想要**离开后再继续上次练习进度，**以便**灵活安排学习时间。

**验收标准**：

- [ ] AC-1: 离开后已保存进度保留 15 天。
- [ ] AC-2: 再次进入同一试卷练习时提示"继续/重新开始"。
- [ ] AC-3: 超过 15 天进度自动清除。
- [ ] AC-4: 同一试卷同一用户只能有一个活跃练习进度。多设备场景下，后打开的设备加载服务端已有进度。

**边缘场景**：

- EC-1: 练习中授权到期时立即终止，系统自动保存已完成的作答到后台（用于 AI 日志追溯），学员端不可查看。提示“授权已失效”。

**优先级**：P1

## 3. 模拟考试

### US-03-05: 模拟考试答题

**作为**学员，**我想要**进行整卷模拟考试，**以便**模拟真实考试场景。

**验收标准**：

- [ ] AC-1: 作答中不显示对错、答案、解析。
- [ ] AC-2: 有时长时服务器倒计时，超时自动交卷。
- [ ] AC-3: 无时长时用户手动交卷。
- [ ] AC-4: 退出后可回到同一场考试，剩余时间按服务器计算。
- [ ] AC-5: 允许对同一试卷多次模拟考试。

**边缘场景**：

- EC-1: 交卷时弹窗提示未答题数量。
- EC-2: 模拟考试中授权失效时（自然到期、主动取消、企业停用），系统自动保存当前作答并立即终止，记录状态为 `terminated`。已终止考试不触发评分、不生成报告。
- EC-3: 断网时前端本地暂存，恢复后自动提交。首期不设补提交时间窗口，恢复网络后立即补提交。
- EC-4: 考试会话绑定用户而非设备，切换设备登录后新设备可继续同一场考试。

**优先级**：P0

---

### US-03-06: 模拟考试作答保存

**作为**学员，**我想要**切题时自动保存作答，**以便**避免数据丢失。

**验收标准**：

- [ ] AC-1: 切题时自动保存当前题作答到服务器。
- [ ] AC-2: 主观题提供"保存答案"按钮。
- [ ] AC-3: 保存失败时提示"保存失败，请检查网络"并自动重试。

**优先级**：P0

---

## 4. 报告与错题本

### US-03-07: 模拟考试报告

**作为**学员，**我想要**查看模拟考试的分析报告，**以便**了解薄弱点。

**验收标准**：

- [ ] AC-1: 报告含总分、客观题/主观题得分、用时、每题详情。
- [ ] AC-2: 按题型、大题/模块、知识点统计得分率。
- [ ] AC-3: AI 学习建议作为报告一部分保存在快照中。生成失败时标记“学习建议生成失败”，不影响报告分数。用户可手动点击“重新生成学习建议”。
- [ ] AC-4: 报告快照保存，后续内容变化不影响历史报告。

**依赖**：US-04-01

**优先级**：P0

---

### US-03-08: 模拟考试记录列表

**作为**学员，**我想要**查看历史模拟考试记录，**以便**回顾和跟踪。

**验收标准**：

- [ ] AC-1: 状态：`scoring` / `scoring_partial_failed` / `completed` / `terminated`（试卷下架、授权失效、账号停用等原因中断）。已终止考试不触发评分、不生成报告。
- [ ] AC-2: 默认按考试开始时间倒序。
- [ ] AC-3: 支持按试卷名称搜索、按状态筛选。
- [ ] AC-4: 列表显示试卷名称、科目、考试时间、得分（已完成时）、状态。

**优先级**：P0

---

### US-03-09: 客观题错题本

**作为**学员，**我想要**通过错题本回顾答错的客观题，**以便**针对性复习。

**验收标准**：

- [ ] AC-1: 答错自动加入（单选/多选/判断/自动匹配填空）。
- [ ] AC-2: 支持手动收藏、移除、标记已掌握。
- [ ] AC-3: 按题目维度去重，记录答错次数。
- [ ] AC-4: "已掌握"后再次答错自动恢复为"未掌握"。
- [ ] AC-5: 支持按题型、来源、掌握状态筛选。
- [ ] AC-6: 支持从错题本触发 AI 讲解。
- [ ] AC-7: 授权失效后隐藏，重新授权后恢复。
- [ ] AC-8: 错题本首期不设容量上限，列表分页展示。
- [ ] AC-9: 错题本中的题目如母题被停用，仍保留显示并标记“该题目已停用”，仍可查看和触发 AI 讲解。

**优先级**：P1
