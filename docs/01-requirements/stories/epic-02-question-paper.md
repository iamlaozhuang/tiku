# Epic 02: 题库、试卷与内容管理

> 源文档：[02-question-paper.md](../modules/02-question-paper.md)

---

## 1. 题库管理

### US-02-01: 题目新建与编辑

**作为**内容老师，**我想要**通过结构化表单创建和编辑题目，**以便**建设题库。

**验收标准**：
- [ ] AC-1: 按题型引导表单，包含完整基础字段（题干、选项、答案、解析、评分点等共 16 项，见 02 §2.5）。
- [ ] AC-2: 富文本编辑题干、解析、标准答案，支持插入图片（≤5MB，JPG/PNG/GIF/WebP）和基础表格。
- [ ] AC-3: 题干/解析上限 10000 字符，超出时编辑器提示。
- [ ] AC-4: 可关联材料、知识点、标签。
- [ ] AC-5: 未被已发布试卷引用的题目可编辑；已锁定题目不能编辑只能复制新题。
- [ ] AC-6: 判断题内部存储为正确/错误，学员端显示为 A.正确 / B.错误。后台录入 A/B 时系统自动转换。
- [ ] AC-7: 主观题评分点含"描述"和"分值"两个字段。评分点分值允许 0.5 粒度。首期不做关键词、可接受表述、扣分规则。

**优先级**：P0

---

### US-02-02: 题目停用与复制

**作为**内容老师，**我想要**停用不再使用的题目或复制已锁定题目，**以便**管控题库质量。

**验收标准**：
- [ ] AC-1: 停用后不能被新试卷选择，已发布试卷中的题目不受影响。
- [ ] AC-2: 已锁定题目可停用。
- [ ] AC-3: 可复制题目为新题（独立编辑）。

**优先级**：P1

---

### US-02-03: 题库搜索与筛选

**作为**内容老师，**我想要**按多维度搜索筛选题目，**以便**快速定位和组卷。

**验收标准**：
- [ ] AC-1: 支持按专业、等级、科目、题型、状态筛选。
- [ ] AC-2: 支持按关键字搜索题干文本。
- [ ] AC-3: 支持按标签和知识点筛选。
- [ ] AC-4: 组卷选题时默认按当前试卷专业/等级/科目预筛选。

**优先级**：P0

---

### US-02-04: 多选题评分规则配置

**作为**内容老师，**我想要**为多选题配置评分规则，**以便**灵活控制评分策略。

**验收标准**：
- [ ] AC-1: 支持"全对得分"和"漏选部分得分，错选不得分"两种规则。
- [ ] AC-2: 默认为"全对得分"。

**优先级**：P1

---

### US-02-05: 填空题评分方式配置

**作为**内容老师，**我想要**选择填空题的评分方式，**以便**区分自动匹配和 AI 评分。

**验收标准**：
- [ ] AC-1: 支持"自动匹配"和"AI 评分"两种方式。
- [ ] AC-2: 切换时保留已录入答案，不自动清除。
- [ ] AC-3: 切换为 AI 评分时需补充评分点；切换为自动匹配时需确认标准答案。
- [ ] AC-4: 报告中标明评分方式。
- [ ] AC-5: 填空题评分方式是母题固有属性，题目加入试卷后不允许在试卷中切换。如需变更，回题库修改母题或复制新题。

**优先级**：P1

---

## 2. 材料管理

### US-02-06: 材料库管理

**作为**内容老师，**我想要**独立维护材料库，**以便**材料可跨题目和试卷复用。

**验收标准**：
- [ ] AC-1: 材料含标题、正文（富文本，上限 30000 字符）、表格、图片、专业/等级/科目、状态。
- [ ] AC-2: 材料仅在被已发布试卷引用后才锁定。锁定后需复制新材料编辑。
- [ ] AC-3: 可停用旧材料。停用后已发布试卷中引用该材料的题目不受影响（使用快照），学员端正常展示。
- [ ] AC-4: 可查看材料被引用的题目和试卷列表。

**优先级**：P0

---

## 3. 组卷与试卷管理

### US-02-07: 草稿试卷组卷

**作为**内容老师，**我想要**从题库选题组卷，**以便**构建完整试卷。

**验收标准**：
- [ ] AC-1: 题目加入草稿时复制当时快照（含关联材料）。
- [ ] AC-2: 草稿内可调整题目顺序、设置分值、调整大题/题组归属。
- [ ] AC-3: 不允许编辑题干、选项、客观题答案和解析。
- [ ] AC-4: 主观题评分点可按试卷调整（增删改），不反向修改母题。
- [ ] AC-5: 大题分值由子题自动累加。

**依赖**：US-02-01, US-02-06

**优先级**：P0

---

### US-02-08: 试卷发布与强校验

**作为**内容老师，**我想要**发布试卷时系统自动校验完整性，**以便**确保试卷质量。

**验收标准**：
- [ ] AC-1: 每道题必须设置分值。
- [ ] AC-2: 总分 = 所有计分子题分值合计。
- [ ] AC-3: 主观题评分点分值合计 = 该题满分。
- [ ] AC-4: 多空填空每空分值合计 = 题目总分。
- [ ] AC-5: 至少 1 道计分题目。不额外限制最少主观题/客观题数量，也不设置最少总分限制。
- [ ] AC-6: 不允许空大题/模块。
- [ ] AC-7: 校验不通过不能发布。
- [ ] AC-8: 发布后内容锁定，被引用的母题锁定。

**优先级**：P0

---

### US-02-09: 试卷下架与删除

**作为**内容老师，**我想要**下架或删除不再使用的试卷，**以便**管控内容。

**验收标准**：
- [ ] AC-1: 下架后学员不能新开始练习/考试。
- [ ] AC-2: 未完成的练习/考试立即终止。
- [ ] AC-3: 已完成的历史报告保留可查看。
- [ ] AC-4: 草稿且无引用可删除；已发布/有记录的试卷不可删除。

**优先级**：P0

---

### US-02-10: 复制试卷

**作为**内容老师，**我想要**复制已发布/已下架试卷为新草稿，**以便**快速创建相似试卷。

**验收标准**：
- [ ] AC-1: 复制时题干、选项、客观题答案和解析从母题重新取当前版本快照；主观题评分点从源试卷中复制（保留源试卷中调整过的评分点描述和分值），而非从母题重新取默认评分点。
- [ ] AC-2: 名称追加"（副本）"后缀，状态为草稿。
- [ ] AC-3: 源试卷中已停用母题仍纳入，标记提示"该题目已停用"。

**优先级**：P1

---

### US-02-11: 试卷元数据与原始文件

**作为**内容老师，**我想要**维护试卷元数据并绑定原始文件，**以便**追溯和管理。

**验收标准**：
- [ ] AC-1: 基础字段（名称、专业、等级、科目、总分）必填，时长可选（10~300 分钟）。
- [ ] AC-2: 扩展元数据（类型、年份、来源等）可选。
- [ ] AC-3: 可绑定多个原始文件（试卷/材料/答案/解析等），标记用途。
- [ ] AC-4: 学员端不展示原始文件。

**优先级**：P1
