## 1. 专业、等级、科目

首期专业：

- 专卖
- 营销
- 物流

物流目前资料为空，但系统分类预留。

等级：

- 一级至五级均可支持。
- 现有资料重点为三级、四级、五级。

科目固定两个值：

- 理论
- 技能

一套试卷必须属于一个科目，不支持理论和技能混合卷。

## 2. 题型

首期题型：

- 单选题
- 多选题
- 判断题
- 填空题
- 简答题
- 案例分析题
- 计算题

材料题组/组合题不是题型，而是结构容器。

材料题组下允许混合题型：

- 单选
- 多选
- 判断
- 填空
- 简答
- 案例分析
- 计算题

操作实务题首期不单独作为题型，可作为题目标签或业务分类。
- **注：题目基础属性中增加 tags（自定义标签列表）字段，供内容老师随意打标（如：操作实务、高频易错等）。**

### 2.1 判断题

系统内部统一存储为正确/错误。

学员端显示为：

- A. 正确
- B. 错误

答案卷或后台录入如果使用 A/B，系统转换为内部正确/错误。

### 2.2 多选题评分规则

多选题每题可配置评分规则。

首期支持两种：

1. 全对得分：错选、漏选、多选均不得分。
2. 漏选部分得分，错选不得分：每个正确选项按比例给分，有错误选项则 0 分。

默认规则为"全对得分"。

### 2.3 填空题评分

填空题分情况评分：

- 短文本填空：按标准答案精确匹配，支持多个可接受答案。
- 多空填空：每空独立配置答案和分值。
- 表述可能多样、需要解释的填空：走 AI 评分。

后台录题时由老师选择评分方式：

- 自动匹配。
- AI 评分。

报告中标明评分方式。

评分方式切换：

- 切换评分方式时保留已录入的答案数据，不自动清除。
- 切换为 AI 评分时需补充评分点；切换为自动匹配时需确认标准答案。
- 后台提示切换影响，由老师确认。

试卷中不可覆盖：

- 填空题评分方式（自动匹配 / AI 评分）是题目的固有属性，在母题中设定。
- 题目加入试卷后，评分方式跟随母题快照，不允许在试卷中切换。
- 如需变更评分方式，需回到题库修改母题（未锁定时）或复制新题修改。

### 2.4 主观题评分点

主观题包括：

- 简答题。
- 案例分析题。
- 计算题中需要文字说明或步骤评分的题。
- 使用 AI 评分的填空题。

首期评分点字段采用最小结构：

- 评分点描述。
- 分值。

不做：

- 关键词。
- 可接受表述。
- 扣分规则。
- AI 评分提示。

AI 评分依据：

- 题干。
- 材料。
- 标准答案/参考答案。
- 解析。
- 评分点描述。
- 每个评分点分值。
- RAG 命中资料。

评分点允许部分得分：

- 每个评分点可给 0 到该评分点满分之间的分数。
- 最小粒度为 0.5 分。
- 题目得分为评分点得分合计。
- 得分不得超过题目满分。
- 未作答主观题直接 0 分，不调用 AI。

分数展示：

- 单题分值允许 0.5。
- 评分点分值允许 0.5。
- 试卷总分允许 0.5。
- 报告总分允许 0.5。
- 展示最多保留 1 位小数；整数不强制显示 `.0`。

### 2.5 题目基础字段

每道题库题目包含以下基础字段：

- 题干（富文本）。
- 题型。
- 专业。
- 等级。
- 科目。
- 选项列表（选择题，每个选项含标号和内容）。
- 标准答案（客观题）/ 参考答案（主观题）。
- 解析（富文本）。
- 评分点列表（主观题）。
- 评分方式（填空题：自动匹配 / AI 评分）。
- 多选评分规则（多选题：全对得分 / 漏选部分得分）。
- 关联材料（可选）。
- 关联知识点列表。
- 标签列表（tags）。
- 状态（可用 / 停用）。
- 来源试卷（可选，录入时关联的原始试卷，用于追溯）。

## 3. 题库、材料与试卷模型

### 3.0 首期录入方式

首期内容建设以人工结构化录入为主，后续再扩展 PDF/Word AI 解析导入。

内容后台采用结构化编辑体验：

- 先维护题库题目、材料，再组卷。
- 通过表单和富文本编辑题干、选项、答案、解析、评分点。
- 通过试卷结构维护大题/模块、材料题组、题目顺序和分值。
- 不做 PDF/Word 左右对照录入器。
- 不做类 Word 整卷编辑后再标注题目、材料、答案、评分点的录入器。
- 原始资料文件可以作为附件绑定和下载，用于人工录入时对照，但不要求首期在线预览和定位联动。

### 3.1 题库优先

首期采用题库优先：

1. 先创建题库题目。
2. 再从题库选择题目组卷。

真题录入时也可以按试卷顺序录题，但系统底层仍形成题库题目，再挂入试卷。

题目状态：

- 可用。
- 停用。

题目不做完整发布流程。

停用题目：

- 不能再被新试卷选择。
- 已经进入已发布试卷的题目不受影响。

题库搜索与筛选：

- 内容后台题库列表支持按专业、等级、科目、题型、状态筛选。
- 支持按关键字搜索题干文本。
- 支持按标签筛选。
- 支持按知识点筛选（选择知识点树节点后显示绑定该节点的题目）。
- 组卷选题时使用相同筛选器，默认按当前试卷的专业/等级/科目预筛选。

### 3.2 题库母题锁定规则

题库母题只有在被已发布试卷引用后才锁定。

规则：

- 未被已发布试卷引用：可以编辑。
- 仅被草稿试卷引用：仍可编辑。
- 草稿试卷里已经加入的题目快照不自动同步更新。
- 如果老师希望草稿试卷使用修改后的母题，需要在草稿试卷中移除旧题，再重新选择。
- 试卷发布后，被该发布试卷引用的母题锁定。
- 锁定后的母题不能编辑、不能删除，只能复制新题后修改。
- 旧题可停用，避免继续被新试卷选择。

草稿试卷不提示母题已更新，也不自动同步。

### 3.3 题目进入试卷

题目加入草稿试卷时复制当时快照。

草稿试卷内：

- 不允许编辑题干。
- 不允许编辑选项。
- 不允许编辑客观题答案。
- 不允许编辑客观题解析。
- 允许调整题目顺序。
- 允许设置分值。
- 允许调整大题/材料题组归属。
- 主观题评分点从母题复制到试卷中，可按试卷调整。

题目分值在试卷中维护。

同一道题在不同试卷可以有不同分值。

### 3.4 客观题答案与解析

客观题答案和解析不允许在试卷中覆盖母题。

如果客观题答案或解析有问题：

- 母题未锁定时，回到题库修改母题。
- 母题已锁定时，复制新题修改。
- 草稿试卷中替换为新题。
- 已发布试卷不可修改，只能下架或复制新卷。

### 3.5 主观题评分规则

题库母题维护默认评分点。

组卷时：

- 默认评分点复制到试卷题目评分规则。
- 试卷中可以根据该卷分值调整评分点分值。
- 可增删评分点。
- 可修改评分点描述。
- 调整只影响该试卷，不反向修改母题。

发布后：

- 试卷内评分规则锁定。
- AI 评分使用试卷内评分规则。
- 考试报告保存评分规则快照。

### 3.6 材料库

材料/案例作为题库里的独立材料对象维护，可复用。

材料字段建议：

- 材料标题。
- 正文。
- 表格。
- 图片。
- 专业。
- 等级。
- 科目。
- 来源资料。
- 状态。

规则：

- 题目可以不关联材料。
- 只有被标记为材料题组/案例材料题时才要求材料。
- 理论题通常不关联材料，但系统允许。
- 技能题可以关联材料，也可以不关联材料。
- 材料可以被多道题、多套试卷复用。
- 材料仅在被已发布试卷（或其中的题目）引用后才锁定。未发布前（或仅在草稿试卷中）允许编辑修改。
- 题目加入草稿试卷时，关联材料一并快照。草稿试卷中的材料内容不随题库材料修改自动同步。如需使用修改后的材料，需在草稿试卷中移除旧题并重新选择。
- 如需修改已锁定材料，复制新材料后编辑。
- 旧材料可停用。
- 材料停用后，已发布试卷中引用该材料的题目不受影响（使用快照），学员端正常展示。停用仅影响后台新组卷时的选择。
- 组卷选材料时默认按试卷的专业/等级/科目筛选。允许跨专业/等级选择材料，但后台显示不匹配提示。发布校验不拦截材料与试卷的专业/等级不匹配，只做提示。

### 3.7 大题、模块、材料题组

现有资料中的理论卷和技能卷都存在结构：

- 理论卷：单项选择题、多项选择题、判断题。
- 技能卷：证件管理、市场管理、案件查办、内部监管等业务模块。

首期保留大题/模块结构。

规则：

- 大题/模块作为结构容器。
- 大题名称、说明、顺序可维护。
- 大题/模块本身不手动设置分值。
- 大题分值由其下所有子题自动累加。
- 报告中可以按大题/模块统计得分率。
- 材料题组本身不计分，只由子题计分。
- 材料题组可显示小计，小计由系统计算。

### 3.8 试卷元数据

试卷除专业、等级、科目、授权、发布状态外，还需要维护以下元数据字段，用于内容管理、筛选和追溯：

基础字段（必填）：

- 试卷名称。
- 专业。
- 等级。
- 科目（理论 / 技能）。
- 总分。
- 考试时长（可选）。

扩展元数据（可选）：

- 试卷类型：真题 / 模拟题。
- 年份。
- 月份。
- 来源地区。
- 来源机构。
- 来源说明（自由文本，如"2024年全国烟草专卖三级技能鉴定"）。
- 出题依据（自由文本，如"依据2022版教材第三篇"）。
- 生成方式：人工 / AI / 混合。首期全部为人工，后续扩展 AI 出题时使用。

规则：

- 扩展元数据首期均为可选，不影响发布校验。
- 内容后台试卷列表支持按试卷名称、专业、等级、科目、状态、试卷类型、年份筛选。默认按更新时间倒序排列。
- 这些字段用于后续去重、统计和检索，首期录入时建议尽量填写。

### 3.9 试卷原始文件绑定

首期试卷支持绑定原始文件，用于留档、追溯和人工录入对照。

可绑定文件类型：

- 试卷文件。
- 材料卷。
- 答题卷。
- 答案卷。
- 解析文件。
- 来源教材或课件文件。

规则：

- 原始文件作为试卷资源附件保存，不直接等同于结构化题目。
- 一套试卷可以绑定多个原始文件。
- 文件需标记文件用途，如试卷、材料、答题、答案、解析、来源。
- 学员端首期不展示或下载这些原始文件。
- 内容后台可下载绑定文件，用于人工录入和核对。
- 试卷发布后，绑定文件随试卷记录保留；是否允许继续追加附件可在实现时按权限控制，但不得影响已发布试卷内容快照。

### 3.10 图片、表格、公式

根据现有资料，题干、材料、解析需要支持图片和表格。

首期能力：

- 富文本编辑题干、材料、解析、标准答案。
- 支持插入图片。
- 支持基础表格。
- 复杂 Word 表格不要求完美还原，老师可手工整理为表格或图片。
- 学员端移动页面表格支持横向滚动。
- AI/RAG 对表格先按 Markdown 表格文本处理。
- 复杂表格可同时保留图片用于展示。

图片上传限制：

- 单张图片上传大小上限 5MB。
- 支持格式：JPG、PNG、GIF、WebP。
- 单个富文本字段内图片数量首期不做硬限制，但建议不超过 10 张。

公式：

- 首期不做公式编辑器。
- 公式可用图片或普通文本表达。
- 后续可扩展 LaTeX/公式节点。

编辑器：

- 首期编辑器能力要求：基础富文本编辑，支持插入图片、表格、链接、列表。

底层保存：

- 富文本内容保存为 HTML 或编辑器 JSON。
- 同时派生纯文本/Markdown，用于搜索、AI 讲解、知识点推荐、RAG 上下文。
- 图片作为附件保存，富文本引用附件 ID。

内容长度限制：

- 题干富文本上限 10000 字符（含 HTML 标签）。
- 解析富文本上限 10000 字符。
- 材料正文富文本上限 30000 字符。
- 超出上限时编辑器提示，不允许保存。

## 4. 试卷状态与发布规则

试卷状态：

- 草稿。
- 已发布。
- 已下架。

草稿：

- 可编辑试卷基本信息。
- 可调整题目顺序、分值、题组归属。
- 可保存不完整内容。

发布：

- 发布后内容锁定，不允许直接修改。
- 学员端只展示已发布试卷。
- 发布时执行强校验。

下架：

- 已发布试卷允许下架。
- 下架后学员不能新开始练习/考试。
- 未完成的练习/模拟考试立即终止。
- 已完成的历史报告保留可查看，前提是用户当前有对应授权。
- 下架不等于删除。

删除：

- 草稿且无引用可删除。
- 已发布试卷不能删除，只能下架。
- 有任何练习/考试记录的试卷不能删除。
- 发布过或已有作答记录的试卷/题目不允许物理删除。

发布强校验：

- 每道试卷题必须设置分值。
- 试卷总分必须等于所有计分子题分值合计。
- 主观题评分点分值合计必须等于该题满分。
- 多空填空每空分值合计必须等于题目总分。
- 试卷必须至少包含 1 道计分题目。不额外限制最少主观题/客观题数量，也不设置最少总分限制。
- 不允许存在无子题的空大题/模块。草稿阶段允许临时存在空大题，发布时校验拦截。
- 校验不通过不能发布。

多试卷快照说明：

- 同一母题可同时被多个草稿试卷引用。各试卷独立快照，不自动同步。
- 如母题在某试卷快照后被修改，其他引用该母题的草稿试卷不会自动更新快照。这是预期行为，确保各试卷编辑独立性。

试卷时长：

- 可选。
- 有时长时模拟考试按服务器时间倒计时。
- 无时长时模拟考试不倒计时，用户手动交卷。
- 练习模式不受时长限制。
- 试卷时长以分钟为单位，正整数。最小值 10 分钟，最大值 300 分钟。不支持秒级精度。

### 4.2 复制试卷

已发布或已下架试卷可复制为新草稿。

规则：

- 复制时题干、选项、客观题答案和解析从母题重新取当前版本快照。主观题评分点从源试卷中复制（保留源试卷中调整过的评分点描述和分值），而非从母题重新取默认评分点。
- 新草稿与源试卷无关联，修改不影响源试卷。
- 复制后的试卷元数据继承源试卷，但名称追加“（副本）”后缀，状态为草稿。
- 如果源试卷中有已停用的母题，复制时该题仍纳入（从母题取最新快照），但在新草稿中标记提示“该题目已停用”。
