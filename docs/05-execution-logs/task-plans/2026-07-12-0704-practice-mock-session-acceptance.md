## 环境门禁

- localhost 必须由显式进程级 0704DB override 启动；不得修改 `.env.local`。
- 开始写入前以脱敏核心角色登录矩阵确认数据库目标，不以 `/login` 健康或单个学员登录代替目标证明。
- 私有账号目录仅在内存读取；凭证、cookie、session、token、DB URL 和 env 值不得进入聊天、日志、截图、证据或仓库。
- `AI_PROVIDER_ENABLED` 保持关闭；不得调用 AI 生成、学习建议重试、Provider 调试、payload 检查或 Cost Calibration。

## 产品 API 流程

1. 用 `personal_standard_student` 获取有效学习范围，并动态选择允许的已发布试卷；不依赖历史硬编码标识。
2. 创建或恢复练习会话，经试卷快照动态选择可作答题目，保存一条合法答案并重新读取验证持久化。
3. 创建或恢复可写模拟考试会话；必要时仅通过产品 API 终止临近截止的旧会话，再保存答案、提交考试并生成或读取报告。
4. 用第二个学员验证无法读取、作答或提交第一学员的练习/模拟考试，也无法读取其报告。
5. 验证未登录请求与管理员角色不能使用学员写入合同；标准版和高级版学员均保留普通学习入口。
6. 用真实浏览器复核练习、模拟考试和报告页面的已保存状态、导航及控制台/网络错误；不截图、不持久化 raw DOM，定位器所需可访问性快照仅在浏览器工具内瞬时使用。
