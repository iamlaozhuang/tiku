# Phase 28 Owner Role Scenario Scripts Plan

## Scope

Write scenario scripts for owner walkthroughs using existing evidence only.

## Student Scripts

- Login/session.
- Authorization/redeem.
- Home.
- Practice.
- Mock exam.
- Exam report.
- Mistake book.
- AI explanation and AI hint paths.

## Admin Scripts

- User management.
- Organization/employee.
- Org authorization.
- Redeem code.
- Question bank.
- Material.
- Paper.
- Knowledge node.
- Resource/knowledge base.
- Model config.
- Audit log.
- AI call log.

## Acceptance Style

Each script records:

- purpose;
- prerequisite data;
- walkthrough steps;
- expected owner-visible result;
- evidence basis;
- limitation label.

## Boundaries

Do not execute the scripts in this task. Do not capture credentials or sensitive data.
