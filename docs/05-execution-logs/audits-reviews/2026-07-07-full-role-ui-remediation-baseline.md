- 本轮是 UI/UX 基线，不是权限安全复测；权限边界结论仍以前序自动化边界巡检为准。
- 截图为当前本地服务状态，若 0704 DB 内容变化，页面密度和空状态会变化。
- `super_admin` 组织后台问题需要结合代码和需求确认：可能是入口策略、组织上下文模型或页面拒绝态文案问题。
