# 0704 AI 生成关闭状态合同对抗式审查

## 结论

通过。状态合同只回答能否发起生成，不携带 Provider、环境、凭证或运行配置；UI 与提交函数双重 fail-closed。Module Run v2 已通过，Cost Calibration 保持阻断。

## 审查矩阵

| 风险面     | 对抗场景                                                                 | 结论                                                                     |
| ---------- | ------------------------------------------------------------------------ | ------------------------------------------------------------------------ |
| 未登录     | 匿名直接读取 availability                                                | 返回标准 401 envelope，不返回状态或配置细节。                            |
| 内容角色   | 内容管理员在关闭状态点击 AI出题/AI组卷                                   | 按钮提交前禁用，事件不产生生成 POST。                                    |
| 组织角色   | 高级组织管理员在关闭状态进入企业 AI 页面                                 | 与内容角色使用同一状态合同；组织请求路径不触发 POST。                    |
| 学习者角色 | 个人高级学员与高级企业员工尝试生成                                       | 两种授权上下文均显示关闭状态并零 POST。                                  |
| 版本边界   | 标准个人或标准组织上下文直接访问                                         | 继续显示授权 unavailable，不调用 availability，不获得高级能力。          |
| 异常状态   | 状态接口失败、缺字段或未知枚举                                           | 页面显示通用状态不可用，保持禁用，不猜测 available。                     |
| 绕过       | 通过事件或函数路径绕过 disabled                                          | 两个 submit handler 在任何 session/网络动作前检查 available 并直接返回。 |
| 信息泄漏   | 状态响应或页面输出 Provider、env、credential、payload、Prompt、AI output | 响应字段白名单只有 `generationAvailability`；页面仅显示业务文案。        |
| 正式内容   | availability 使 AI 结果直接进入正式题目/试卷                             | 未修改生成 POST、评审、采用、发布或正式题源合同；原边界回归通过。        |
| AI组卷题源 | 状态合同扩大本地选题范围                                                 | 未修改组卷装配或题源代码；B1+B2 累计回归通过。                           |

## 品味复核

- API 使用标准 `{ code, message, data }` 与 camelCase 字段。
- 页面使用既有 design tokens、紧凑状态区和明确的禁用语义，无硬编码颜色或新依赖。
- 状态合同独立于生成 payload，不建立万能配置接口，不把服务端授权降级为客户端显隐。
- 新路由认证后才返回脱敏枚举；未知状态一律关闭。

## 残余风险

- 当前 localhost 未挂载原 0704DB，四角色真实会话与截图验收未在本批伪造执行；保留到环境恢复后的累计验收。
- `available` 分支只由单元合同验证，未执行 Provider-enabled 行为；这是本轮明确禁止边界，不是 release readiness 证据。
